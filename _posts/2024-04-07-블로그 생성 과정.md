# 블로그 생성 과정

1. Ruby 설치 (3.1.x - x86 버전)
2. git bash 를 이용하여 bundle 및 jekyll 설치

- 설치
```shell
$ gem install jekyll
$ gem install bundle
```

- 버전 확인 (설치 확인)

```shell
$ jekyll -v
$ bundle -v
```

## 소감

그래서 Github Action 으로 속성을 바꾸어 주고 Configure를 진행!
